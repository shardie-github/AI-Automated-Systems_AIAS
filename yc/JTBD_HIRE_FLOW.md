# Jobs-to-Be-Done Hire Flow — AIAS Platform

**Last Updated:** 2025-01-29  
**Purpose:** Map user journey from awareness to "hire" (signup → activation → value)

---

## Hire Flow Overview

**Journey:** Awareness → Consideration → Signup → Activation → Value → Ongoing Use

---

## Stage 1: Awareness

### How Users Become Aware

**Channel 1: Shopify App Store**
- User searches "order automation"
- Sees AIAS Platform listing
- Reads description: "Automate order processing, save 10+ hours/week"
- **Job Becomes Obvious:** "This can automate my order processing"

**Channel 2: SEO/Content Marketing**
- User searches "Shopify automation Canada"
- Finds AIAS Platform blog post or landing page
- Reads about order automation solution
- **Job Becomes Obvious:** "This can solve my order processing problem"

**Channel 3: Referral**
- Friend refers user via referral link
- User clicks referral link
- Sees personalized landing page
- **Job Becomes Obvious:** "My friend uses this for order automation"

### Moments Where Job Becomes Obvious
- **Shopify App Store:** When user reads "Automate order processing"
- **Landing Page:** When user sees "Save 10+ hours/week"
- **Referral:** When user sees friend's success story

## Stage 2: Consideration

### How Users Consider AIAS Platform

**Process:**
1. User reads about AIAS Platform
2. Compares vs. alternatives (Zapier, Make, manual)

5. Decides to sign up

- **Pricing Page:** When user sees "$49/month unlimited vs. $50/month for 5"
- **Integrations Page:** When user sees Canadian integrations (Shopify, Wave, RBC, TD)
- **Comparison:** When user compares vs. alternatives

## Friction Points Summary

### High Priority Friction Points
1. **Landing Page:** Doesn't clearly explain Canadian integrations
2. **Onboarding:** Too many steps (5 steps vs. 3 steps)
3. **Workflow Builder:** Needs more guidance (tooltips, tutorials)

### Medium Priority Friction Points
1. **Signup Form:** Too long (3 fields vs. 1 field)
2. **Notifications:** Too frequent (need digest emails)
3. **Upgrade Prompts:** Too aggressive (need subtle triggers)

---

## Opportunities Summary

### High Priority Opportunities
1. **Add Video Demo:** Show order automation in action
2. **Reduce Onboarding Steps:** 5 steps → 3 steps
3. **Add Guided Tour:** Help users create first workflow

### Medium Priority Opportunities
1. **Add Social Signup:** Google, Shopify OAuth
2. **Add Digest Emails:** Daily summary instead of per-execution
3. **Add Usage-Based Triggers:** Show upgrade when approaching limits
