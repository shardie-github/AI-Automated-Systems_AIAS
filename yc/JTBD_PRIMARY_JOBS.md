## Job Format

**Format:** "When [situation], I want to [motivation], so I can [outcome]"

---

## Primary Jobs

### Job 1: Automate Order Processing

**Statement:** "When I get a new Shopify order, I want to automatically send order confirmation and shipping label, so I can save 3 hours/day and focus on growing my business."

**Who:** Sarah Chen, Shopify store owner  
**Situation:** New Shopify order comes in  
**Motivation:** Automatically send confirmation and shipping label  
**Outcome:** Save 3 hours/day, focus on growth

**Current Flow:**
1. Order comes in via Shopify
2. Manually send order confirmation email
3. Manually generate shipping label
4. Manually update inventory
5. Manually track order status

**Desired Flow:**
1. Order comes in via Shopify
2. Workflow automatically triggers
3. Order confirmation email sent automatically
4. Shipping label generated automatically
5. Inventory updated automatically
6. Order status tracked automatically

**Time Saved:** 3 hours/day = 15 hours/week = 60 hours/month

---

### Job 2: Generate Proposals Automatically

**Statement:** "When I need to write a proposal for a client, I want AI to draft it based on my templates, so I can win more clients and work less."

**Who:** Mike Thompson, Business consultant  
**Situation:** Need to write proposal for client  
**Motivation:** AI drafts proposal based on templates  
**Outcome:** Win more clients, work less

**Current Flow:**
1. Client requests proposal
2. Manually write proposal (2 hours)
3. Review and edit proposal
4. Send proposal to client
5. Follow up manually

**Desired Flow:**
1. Client requests proposal
2. AI drafts proposal based on templates (5 minutes)
3. Review and edit proposal
4. Send proposal to client automatically
5. Follow up automatically

**Time Saved:** 2 hours → 5 minutes = 1.75 hours saved per proposal

**Who:** Jessica Martinez, Real estate agent  
**Situation:** New lead comes in  
**Motivation:** AI pre-qualifies and schedules showings  
**Outcome:** Close 2x more deals

**Current Flow:**
1. Lead comes in via website/form
2. Manually qualify lead (1 hour)
3. Manually schedule showing
4. Manually follow up
5. Manually track lead status

**Desired Flow:**
1. Lead comes in via website/form
2. AI pre-qualifies lead automatically
3. AI schedules showing automatically
4. AI follows up automatically
5. Lead status tracked automatically

**Time Saved:** 1 hour per lead = 30 hours/week for 30 leads

### Job 4: Sync Inventory Across Platforms

**Statement:** "When inventory changes in Shopify, I want it to automatically sync to Wave Accounting and other platforms, so I can avoid manual data entry errors."

**Who:** Sarah Chen, Shopify store owner  
**Situation:** Inventory changes in Shopify  
**Motivation:** Automatically sync to Wave Accounting  
**Outcome:** Avoid manual data entry errors

**Current Flow:**
1. Inventory changes in Shopify
2. Manually update Wave Accounting
3. Manually update other platforms
4. Risk of errors (wrong quantities, prices)

**Desired Flow:**
1. Inventory changes in Shopify
2. Workflow automatically syncs to Wave Accounting
3. Workflow automatically syncs to other platforms
4. No manual errors

**Time Saved:** 2 hours/week on manual data entry

### Job 5: Send Customer Follow-Up Emails

**Statement:** "When a customer makes a purchase, I want to automatically send follow-up emails (thank you, review request, upsell), so I can increase customer lifetime value."

**Who:** Sarah Chen, Shopify store owner  
**Situation:** Customer makes purchase  
**Motivation:** Automatically send follow-up emails  
**Outcome:** Increase customer lifetime value

**Current Flow:**
1. Customer makes purchase
2. Manually send thank you email
3. Manually send review request email
4. Manually send upsell email
5. Risk of forgetting or sending wrong emails

**Desired Flow:**
1. Customer makes purchase
2. Workflow automatically sends thank you email
3. Workflow automatically sends review request email (after 7 days)
4. Workflow automatically sends upsell email (after 30 days)
5. No manual work, consistent messaging

**Time Saved:** 1 hour/week on manual email sending
